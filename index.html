🚀 Quick Start

1. Get API Keys

Provider Sign Up Best For
DeepSeek platform.deepseek.com Best balance of speed & quality
OpenAI platform.openai.com GPT-4o, GPT-5, o3, o4-mini
Anthropic console.anthropic.com Claude Sonnet 4, Opus 4, Fable 5
Google Gemini aistudio.google.com Gemini 2.5 Pro (free tier)
NVIDIA NIM build.nvidia.com Free tier available
OpenRouter openrouter.ai 200+ models through one API

2. Start the API Server

bun local-api-server.js

3. Launch AIGENEV7

CODEBUFF_API_KEY=local-dev-key CODEBUFF_APP_URL=http://localhost:3457 ./aigenev7.exe
